# Claude Code Assist

Claude Code runs as an AI coding agent within the Docker network.

## Tips for AI Agents

- Claude Code can generate, review, and debug code across multiple languages.
- Supports multi-file editing with context awareness.
- Use for code reviews, refactoring, and test generation.
- Integrates with Git for version-controlled workflows.
