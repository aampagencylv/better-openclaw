# Coolify Deployment

Coolify is available at `http://{{COOLIFY_HOST}}:{{COOLIFY_PORT}}` within the Docker network.

## Tips for AI Agents

- Coolify is a self-hosted Heroku/Netlify alternative.
- Deploy from Git repos, Docker images, or Dockerfiles.
- Supports automatic SSL, databases, and preview deployments.
- Use webhooks for CI/CD integration.
