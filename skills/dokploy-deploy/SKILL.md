# Dokploy Deployment

Dokploy is available at `http://{{DOKPLOY_HOST}}:{{DOKPLOY_PORT}}` within the Docker network.

## Tips for AI Agents

- Dokploy is a self-hosted PaaS similar to Vercel/Heroku.
- Supports deployments from Git, Docker, and compose files.
- Automatic SSL certificates and custom domains.
- Built-in database provisioning for PostgreSQL, MySQL, and Redis.
