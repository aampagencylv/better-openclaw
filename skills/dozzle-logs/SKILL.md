# Dozzle Log Viewer

Dozzle is available at `http://{{DOZZLE_HOST}}:{{DOZZLE_PORT}}` within the Docker network.

## Tips for AI Agents

- Real-time log streaming for all running Docker containers.
- Search and filter logs across multiple containers simultaneously.
- No storage required — Dozzle streams directly from Docker.
- Supports log level highlighting and JSON log formatting.
