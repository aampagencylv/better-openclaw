# Beszel Monitor

Beszel is available at `http://{{BESZEL_HOST}}:{{BESZEL_PORT}}` within the Docker network.

## Tips for AI Agents

- Beszel provides lightweight server monitoring with CPU, memory, disk, and network metrics.
- Deploy the Beszel agent on each server to monitor.
- Access the dashboard for real-time and historical charts.
- Supports alerting via webhooks on threshold breaches.
