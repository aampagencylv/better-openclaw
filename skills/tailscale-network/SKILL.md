# Tailscale Network

Tailscale creates a secure mesh VPN connecting all your services.

## Tips for AI Agents

- Tailscale creates encrypted peer-to-peer connections between nodes.
- Access remote services as if they were on the local network.
- MagicDNS provides human-readable hostnames for all connected devices.
- ACL policies control which services can communicate.
