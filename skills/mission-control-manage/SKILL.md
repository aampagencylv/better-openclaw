Mission Control is available at `http://{{MISSION_CONTROL_HOST}}:{{MISSION_CONTROL_PORT}}` within the Docker network.

## Tips for AI Agents

- Mission Control provides a dashboard for managing multiple AI agents.
- Monitor agent health, task queues, and resource usage.
- Configure agent routing and load balancing.
- View logs and metrics for all connected agents.
