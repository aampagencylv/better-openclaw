# OpenCode Development

OpenCode provides an open-source AI coding assistant.

## Tips for AI Agents

- OpenCode supports code completion, generation, and refactoring.
- Works with local models via Ollama for privacy-first development.
- Terminal-based interface for seamless CLI workflows.
- Supports multiple programming languages and frameworks.
